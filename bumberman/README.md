# 🧨 범버맨 게임 (Bomberman Game)

크레이지 아케이드 스타일의 웹 기반 범버맨 게임입니다.

## 🎮 게임 소개

클래식한 범버맨 게임을 HTML5 Canvas와 JavaScript로 구현했습니다. 미로 같은 맵에서 폭탄을 설치하여 적들을 제거하고 승리하는 것이 목표입니다.

## 🚀 실행 방법

1. 프로젝트를 클론하거나 다운로드합니다
2. `index.html` 파일을 웹 브라우저에서 엽니다
3. 게임이 자동으로 시작됩니다!

## 🎯 게임 목표

- 🔴 모든 빨간색 적 캐릭터를 제거하세요
- 💀 적이나 폭발에 닿지 않도록 주의하세요
- 🏆 모든 적을 제거하면 승리합니다!

## 🕹️ 조작법

| 키 | 동작 |
|---|---|
| ⬆️ 방향키 위 (또는 W) | 위로 이동 |
| ⬇️ 방향키 아래 (또는 S) | 아래로 이동 |
| ⬅️ 방향키 왼쪽 (또는 A) | 왼쪽으로 이동 |
| ➡️ 방향키 오른쪽 (또는 D) | 오른쪽으로 이동 |
| 🚀 스페이스바 | 폭탄 설치 |
| 🔄 F5 | 게임 재시작 |

## 🎪 게임 요소

### 캐릭터
- 🔵 **플레이어**: 파란색 원형 캐릭터 (당신이 조종)
- 🔴 **적**: 빨간색 원형 캐릭터 (AI가 조종, 랜덤 이동)

### 맵 요소
- 🟫 **벽**: 갈색 고정 벽 (파괴 불가)
- 📦 **블록**: 연갈색 파괴 가능한 블록
- 💣 **폭탄**: 일정 시간 후 십자 모양으로 폭발
- 💥 **폭발**: 폭탄 폭발 효과 (플레이어와 적 모두에게 피해)

### 아이템 시스템
블록을 파괴하면 30% 확률로 아이템이 드랍됩니다:

| 아이템 | 효과 |
|--------|------|
| 🟡 B | **폭탄 개수 증가**: 동시에 설치할 수 있는 폭탄 수 +1 |
| 🟡 F | **폭탄 위력 증가**: 폭발 범위 +1 |
| 🟡 S | **이동 속도 증가**: 캐릭터 이동 속도 향상 |

## 🏗️ 기술 스택

- **HTML5**: 게임 구조 및 캔버스 설정
- **CSS3**: 스타일링 및 레이아웃
- **JavaScript (ES6)**: 게임 로직 및 렌더링
- **Canvas API**: 2D 그래픽 렌더링

## 🎨 게임 특징

- 📱 **반응형 디자인**: 다양한 화면 크기에서 플레이 가능
- 🤖 **AI 적**: 지능적인 랜덤 이동 패턴
- 💎 **아이템 시스템**: 전략적 플레이를 위한 파워업
- 🎯 **충돌 감지**: 정확한 히트박스 시스템
- 🎪 **부드러운 애니메이션**: 60FPS 게임 루프
